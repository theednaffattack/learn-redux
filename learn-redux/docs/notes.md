## 07 - All About Redux Reducers

- A Reducer handles actions and updates the State

- You can think of them like JAvaScript events where a Reducers is like a listener

- These are kept in the 'reducers' folder

- We create a reducer for each piece of our state

- a Reducer takes two things:
  1) the action (info about what happened)
  2) a copy of the current state
